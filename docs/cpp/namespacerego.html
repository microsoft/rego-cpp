Detailed Description

This namespace provides the C++ API for the library. It includes all the token types for nodes in the AST, the well-formedness definitions for each pass, the built-in system and custom types for handling various kinds of data (e.g. the BigInt class).

Driver & rego::driver ( const BuiltIns &  builtins)

Creates a test driver.

The test driver performs probabilstic testing of the complication. For each pass it generates random AST outputs depending on the well-formed expression of the input, runs them through the pass logic, and then checks it against the wf expression for the output.
