Detailed Description

Contains the C++ API for the Rego library. This namespace provides the C++ API for the library. It includes all the token types for nodes in the AST, the well-formedness definitions for each pass, the built-in system and custom types for handling various kinds of data (e.g. the BigInt class).

◆ version()

Node rego::version ( )

Returns a node representing the version of the library.

The resulting node will be an object containing:

  • The Git commit hash ("commit")
  • The library version ("regocpp_version")
  • The supported OPA Rego version ("version")
  • The environment variables ("env")
